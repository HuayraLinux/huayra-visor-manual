Cambiar aplicaciones predeterminadas

De Huayra

Cuando se pulsa dos veces sobre un archivo, se abre con la aplicación predeterminada para ese tipo de archivos. Se puede abrir con una aplicación distinta, buscar aplicaciones en línea, o establecer la aplicación predeterminada para todos los archivos del mismo tipo. Lo mismo ocurre para los links de los diferentes sitios webs y los correos electrónicos. A continuación encontrarás todo lo que necesitás saber al respecto.

[editar] Cambiar la aplicación predeterminada

Podés cambiar la aplicación predeterminada que utilizás para abrir los archivos de un cierto tipo. Esto te permitirá abrir tu aplicación preferida cuando pulses dos veces sobre un archivo para abrirlo. Por ejemplo, puede que quieras que se abra tu reproductor de música favorito al pulsar dos veces sobre un archivo MP3.

  1. Seleccioná un archivo del tipo cuya aplicación predeterminada querés cambiar. Por ejemplo, para cambiar la aplicación usada para abrir archivos MP3, seleccioná un archivo .mp3.
  2. Pulsá con el botón derecho del mouse y seleccioná Propiedades.
  3. Seleccioná la pestaña Abrir con.
  4. Seleccioná la aplicación que querés y pulsá Establecer como predeterminada. De manera predeterminada, el gestor de archivos solo muestra aplicaciones que sabe que pueden manejar el archivo. Para buscar entre todas las aplicaciones en tu equipo, pulsá Mostrar otras aplicaciones.
    Si Otras aplicaciones contiene una aplicación que quisieras usar a veces, pero que no querés convertir en la predeterminada, seleccioná esa aplicación y pulsá en Añadir. Esto la añadirá a Aplicaciones recomendadas. A partir de entonces podrás usar la aplicación pulsando con el botón derecho sobre el archivo y seleccionándola de la lista.

Esto cambiará la aplicación predeterminada no solo para el archivo seleccionado, sino para todos los archivos del mismo tipo.

[editar] Abrir archivo con otra aplicación con el botón derecho

Para abrir un archivo con una aplicación distinta de la predeterminada, pulsá en el archivo y seleccioná la aplicación que deseás en la parte superior del menú. Si no ves la aplicación que querés, pulsá en Abrir con otra aplicación. De manera predeterminada, el gestor de archivos sólo muestra las aplicaciones que sabe que pueden manejar el archivo. Para buscar todas las aplicaciones en tu equipo, pulsá en Mostrar otras aplicaciones.

Si aún no encontrá la aplicación que querés, podés buscar más aplicaciones pulsando Buscar aplicaciones en línea. El gestor de archivos buscará en línea paquetes que contengan aplicaciones que manejen archivos de ese tipo.

[editar] ¿Con qué navegador abro mis sitios web?

Cuando pulsás en un enlace a una página web, en cualquier aplicación, se abre un navegador web automáticamente en esa página. Si tenés más de un navegador instalado, la página puede no abrirse en el navegador que querías. Huayra viene con un navegador web propio, pero podés descargarte otros navegadores libres (Ver ¿Cómo instalar/desinstalar aplicaciones en Huayra?). Para solucionar este problema, cambiá el navegador predeterminado:

  1. Pulsá en Icono de Configuración del sistema en el Panel Superior a la derecha.
  2. Abrí Aplicaciones preferidas del listado de la categoría Personal del Centro de Control.
  3. Elegí qué navegador web querés vincular para abrir, cambiando la opción Web.

Al abrir un navegador web diferente, te puede decir que no es el navegador predeterminado. Si esto sucede, pulsá en el botón Cancelar (o similar) para que no trate de configurarse como el navegador predeterminado.

[editar] Cambiar qué aplicación de correo electrónico se usa para escribir correos

Al pulsar en un botón o enlace para enviar un nuevo correo electrónico (por ejemplo, en tu procesador de textos), la aplicación de correo predeterminada se abrirá con un mensaje en blanco, listo para escribir. Si tenés más de una aplicación de correo instalada, la aplicación de correo mal podría abrirse. Podés resolver el problema cambiando la aplicación de correo electrónico predeterminada:

  1. Pulsá en Icono de Configuración del sistema en el Panel Superior a la derecha.
  2. Abrí Aplicaciones preferidas del listado de la categoría Personal del Centro de Control.
  3. Elegí qué cliente de correo querés usar de manera predeterminada, cambiando la opción Correo.
